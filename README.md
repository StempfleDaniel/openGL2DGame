# openGL2DGame
# A 2D ghost shooter with sprite animation


## Controlls:
 - **Walking:** WASD
 - **Shooting:** Spacebar
 - **Collision with ghost** closes the game

If the game is running **slowly** readjust the speed by changing `float gameSpeed` in `playground.cpp`:
